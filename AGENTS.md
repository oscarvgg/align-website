# Align Website (`align-website/`)

Marketing website + blog built with Hugo.

## Build
- Generate site: `cd align-website && hugo --cleanDestinationDir`
- Optional local server: `cd align-website && hugo server`

## Content
- Blog posts live in `align-website/Content/blog/` (tone guidelines in that folder’s `AGENTS.md`).
- Help pages live in `align-website/Content/help/` (style guidelines in that folder’s `AGENTS.md`).

## Notes
- `Output/` is generated by Hugo; prefer regenerating over editing HTML by hand.
- Site configuration is in `hugo.toml`, and templates are in `layouts/`.

## Blog Writing (`align-website/Content/blog`)

### Style

- Conversational and relatable tone (Ali Abdaal-inspired).
- Use personal anecdotes where helpful; keep it practical.
- Provide actionable advice; include examples.
- Subtly connect advice to Align features (no hard sell).
- End with a light call-to-action (e.g., try Align / subscribe).

### Suggested Structure

Problem → Solutions → Examples → Align tie-in → Call-to-action

## Help Docs (`align-website/Content/help`)

### Style

- Direct, informational, and scannable.
- Prefer short steps, clear headings, and concrete UI labels.
- Avoid marketing language; optimize for user success.
